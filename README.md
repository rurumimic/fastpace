# Fast Pace

## Usage

### Build

```bash
pip install -U pip build
python -m build
```

### Install

```bash
pip install dist/fastpace-0.0.1-py3-none-any.whl
```

### Run

#### Run a executable

```bash
fastpace
```

#### Run a module

```bash
>>> import fastpace
>>> fastpace.start_app()
```

#### Run FastAPI Dev

```bash
fastapi dev
```
